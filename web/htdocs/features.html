Features

General

  • Supports relatively serialization of XML documents and formatting of XML document output. Supports options for formatting of serialized XML documents and for preserving whitespace in text.
  • Supports multiple views of an XML document and is designed to allow new views to be added easily.
  • Allows opening and editing of multiple XML documents in one open window.

Tree View

  • Supports adding and deleting of Nodes in an XML document using context menus.
  • Supports editing an XML node's name and values in an intuitive way.
  • Supports adding and editing of attributes of an element node.
  • Has preliminary Drag and Drop support.

Source View

  • Supports text editing of the full XML source of any XML document
  • Supports cut, copy, and paste and other limited features.

Planned Features

  • Support for Beanshell plugins.
  • Support for transforming XML documents using XSLT stylesheets.
  • Support for validation of an XML document using DTDs and other forms of Schema.
  • Support for Unlimited Undo
  • Syntax highlighted source view
