Support

Mailing Lists

These three mailing lists will be very low traffic, hopefully not forever tho ;-)
  • jsXe-users: Subscribe/Unsubscribe/Preferences: a general discussion list for users of jsXe.
  • jsXe-devel: Subscribe/Unsubscribe/Preferences: a list for technical discussion regarding jsXe development.
  • jsXe-cvs: Subscribe/Unsubscribe/Preferences: a list where cvs commits are posted. Anyone subscribing to this list will be notified when changes are commited to cvs.

 

Forums

To get support if you're having problems installing or running jsXe please post to the Help forum. Any other comments and criticism can be posted in- the open discussion forum.
  • Open Discussion
  • Help

Reporting Bugs

Reporting bugs is easy. Just submit a bug report on jsXe's Bug Tracker. Just enter a description of the bug and if possible how to reproduce it and it will be assigned and addressed by the project development team.

Feature Requests

Features requests are always welcome and will be addressed based on user need and whether the feature fits with jsXe's vision. Feature requests are easy too! Just submit a feature requests to jsXe's Feature Request Tracker.
