Overview

jsXe is the Java Simple XML Editor.

Although you may pronounce it any way you wish I encourage you to pronounce it "jay sexy". It is written in Java using the Swing toolkit, and Xerces-J and is released under the terms of the GNU General Public License. It aims to provide end users and developers with an intuitive way of creating XML documents that is simple enough to deal with any XML document but flexible to allow the addition of support for XML document formats through the use of plugins.

jsXe is currently in a planning and pre-alpha stage and many intended features are not currently implemented. It is not ready for extensive use and is considered a work in progress. If you require a free XML editor and jsXe does not currently serve your needs, I recommend that you try Pollo or Jaxe.
