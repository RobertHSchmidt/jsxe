Requirements

jsXe requires Java 2 version 1.3 or 1.4. If you aren't sure which version to use please choose 1.4

jsXe currently requires a specific version of Xerces-J to run. Xerces-J 2.3.0 is required since some features are implemented using the DOM level 3 interfaces that were in 2.3.0 but were changed in later versions.

Download

The current version of jsXe is 0.1.43 You can either download jsXe in binary format or you can download the source distribution from sourceforge. The normal distributions have the required Xerces-J libraries packaged with them but you can download binary and source distributions without the libraries if you already have Xerces-J 2.3.0 installed.

Install

To install and run jsXe simply unzip jsXe into the directory of your choosing and run the included batch program or shell script. You will need to make sure that the java runtime is in your path.
