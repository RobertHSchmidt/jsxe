Versioning System

The version of jsXe has three parts. The first part is the major version number. This will be incremented most likely when a new release is incompatible with older releases or when jsXe recieves coveted 1.0 status, i.e. it's status becomes 'Production/Stable'. The second part is the minor version number. This will be incremented when jsXe implements new features and a new stable version is released. The third part will specify the build number. So every time a build is done this will be incremented.

As of yet there is no stable version of jsXe.
