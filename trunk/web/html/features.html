Current Features

Here are some features that jsXe implements.
  • Tree View - jsXe allows you to view an XML document in the form of a tree and edit some nodes in the document. It also allows you to create and edit attributes of element nodes.
  • Source View - jsXe provides a method for veiwing and editing the source of an XML Document.
  • Intuitive interface - jsXe provides an intuitive interface that allows you to create XML documents easily.
