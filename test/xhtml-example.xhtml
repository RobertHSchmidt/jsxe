XHTML Example

Here is an example of a page written in XHTML.

If you view the source for this page, you may spot a few differences e.g.

  • The DTD definition on the top of the page (see XHTML 1.0 for an explanation)
  • All tags should be in lower case for example: <b></b> (this is a University Web Standard)
  • All tags should be closed for example: <p></p> (this is a University Web Standard)
  • Attribute values should be in quotes for example: src="logo.gif" (this is best practice anyway)
  • Tags are correctly nested (this is best practice anyway)

1. The correct DTD must be used. 2. Scripts and styles require extra syntax.
3. Tags must be correctly nested. 4. Tags must be in lower case.
5. All tags must have an end tag. 6. The tags <br>and <hr> must be ended with <br /> and<hr />
7. All attributes must be quoted. 8. Attributes should not contain line
breaks or multiple white space.

For a full explanation see XHTML 1.0 and especially Appendix C. HTML Compatibility Guidelines
